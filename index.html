Solucion de Fuerza Bruta

En este algoritmo, se supone que el prefijo es la primera cadena y a partir de ahi se van haciendo transformaciones con las demas cadenas reduciendo desde el final hasta el inicio hasta llegar hasta la cadena vacia que significaria que no existe un prefijo comun mas largo.

Asi se ira comparando el prefijo contra todas las cadenas que existen en el arreglo y se ira reduciendo la variable del prefijo

const longestCommonPrefix = (cadenas, tam) => {
  if (tam === 0) {
      return "-1";
  }
  let prefijo = cadenas[0];

  for (let i = 1; i < cadenas.length; i++) {
      while (cadenas[i].indexOf(prefijo) != 0) {
          prefijo = prefijo.substring(0, (prefijo.length) - 1)
      }
      if (prefijo.length === 0) {
          return "-1";
      }
  }
  return prefijo;
  };
